## Purpose
This application is a simple IT Helpdesk Ticket Registration System.
It allows students to submit technical issues and assigns a technician
based on the selected priority level.

## Tech Stack
- Python 3
- Modular Programming

## Files
- main.py
- ticket.py
- display.py

## How to Use

1. Run:

```bash
python main.py
```

2. Enter:
   - Student Name
   - Student ID
   - Issue
   - Location
   - Priority Level

3. The program will display the completed helpdesk ticket.
