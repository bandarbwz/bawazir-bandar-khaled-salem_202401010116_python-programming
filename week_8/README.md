# Food Delivery Receipt Generator

## Purpose

This application generates a receipt for a food delivery order. It collects customer information, calculates subtotal, service charge, delivery charge, and displays the final receipt.

## Tech Stack

- Python 3

## How to Use

1. Run `main.py`
2. Enter customer information.
3. The receipt will be displayed automatically.

## Demonstration

Run the program and record the output using a screen recorder or GIF.
